# Bank Account System

## Автор
**Мечева Алёна** 

## Краткое описание
Консольное приложение для управления банковскими счетами с поддержкой различных типов счетов (расчетный, сберегательный, кредитный), транзакций, перевода средств между счетами, фильтрации операций и сохранения данных в JSON.

### Основные возможности:
- Создание счетов трех типов
- Внесение и снятие средств
- Перевод между счетами
- История транзакций в виде очереди
- Фильтрация транзакций по дате, типу и ID счета
- Сохранение/загрузка данных в JSON
- Обработка ошибок (недостаточно средств, лимиты, некорректный ввод)
